基础

  1. 数据包一般有三部分：data + PH + FH
    1. data：可以再分为tcp头 + 主体、udp头 + 主体
    2. PH：Internet packet header。ipv4：2字节头部信息 + 2字节长度 + 1字节协议 + 7字节其他 + 4字节srcIP + 4自己distIP
      • 协议：TCP
      • 协议：UDP
      • 协议：ICMP
    3. FH：LAN frame header。6字节src mac addr + 6字节 dist mac address + 2字节type
      • type=0800：ip帧
      • type=0806：arp帧
