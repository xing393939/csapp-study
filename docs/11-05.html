05.第五章 - 程序优化

第五章 - 程序优化

程序优化

  1. combine1：+=10，*=10
  2. combine2：+=7，*=9。vec_length的计算移到for循环之上。
  3. combine3：+=7，*=9。vec_get_element函数改成直接数组访问并没有提升性能。
  4. combine4：+=1，*=3。用临时变量保存中间结果（临时变量保存在寄存器中）
  5. combine5：+=1.0，*=3.0。acc = (acc + data[i]) + data[i+1]
  6. combine6：+=0.8，*=1.5。acc0 += data[i]; acc1 += data[i+1];（2x2循环展开）
  7. combine7：+=1.0，*=1.5。acc = acc + (data[i] + data[i+1])
  8. combine6：+=0.5，*=1.0。(10x10循环展开)
